Commandos básicos

Iniciar projeto no git
- git init

Status das alterações
- git status

Adicionar arquivo para fazer commit
- git add <nome do arquivo>

Adicionar todos arquivo novos
- git add .

Fazer commit
- git commit -m "Mensagem do commit"

Ver log
- git log

clonar repositório
- git clone <url do repositório>

fazer chackout em nova branch
- git checkout -b <nome da branch>
